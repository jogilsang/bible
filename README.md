### 로마서

10:13 누구든지 주의 이름을 부르는 자는 구원을 받으리라

### 고린도전서

* 1:8 주께서 너희를 우리 주 예수 그리스도의 날에 책망할 것이 없는 자로 끝까지 견고하게 하시리라
